# 数论

## 知识点

### 素数

#### 素数的定义

素数是指大于1的自然数，除了1和它本身外，不能被其他自然数整除的数。换言之，一个数如果只有两个正因数（1和它本身），那么它就是素数。

#### 素数的性质

- 素数有无穷多个（欧几里得证明）。
- 除了2以外，所有素数都是奇数。
- 素数的分布遵循素数定理：小于n的素数个数大约为$n/\ln(n)$。

#### 素数的判断

素数的判断是数论中的基本问题。以下是几种常见的判断方法：

- 朴素题解

朴素题解是通过检查从2到sqrt(n)的所有整数是否能整除n来判断。

``` c++
bool is_prime(int n) {
    if (n <= 1) return false;
    if (n <= 3) return true;
    if (n % 2 == 0 || n % 3 == 0) return false;
    for (int i = 5; i * i <= n; i += 6) {
        if (n % i == 0 || n % (i + 2) == 0) return false;
    }
    return true;
}
```

- 埃拉托斯特尼筛法

埃拉托斯特尼筛法是一种简单高效的素数筛法，用于找出小于等于n的所有素数。其基本思想是从2开始，将每个素数的倍数标记为合数。

``` c++
vector<int> sieve_of_eratosthenes(int n) {
    vector<bool> is_prime(n + 1, true);
    vector<int> primes;
    is_prime[0] = is_prime[1] = false;
    for (int i = 2; i <= n; ++i) {
        if (is_prime[i]) {
            primes.push_back(i);
            for (int j = i * 2; j <= n; j += i) {
                is_prime[j] = false;
            }
        }
    }
    return primes;
}
```

线性筛法

线性筛用于高效筛出素数。

``` c++
vector<int> linear_sieve(int n) {
    vector<bool> is_prime(n + 1, true);
    vector<int> primes;
    vector<int> min_factor(n + 1, 0);
    is_prime[0] = is_prime[1] = false;
    for (int i = 2; i <= n; ++i) {
        if (is_prime[i]) {
            primes.push_back(i);
            min_factor[i] = i;
        }
        for (int p : primes) {
            if (i * p > n) break;
            is_prime[i * p] = false;
            min_factor[i * p] = p;
            if (i % p == 0) break;
        }
    }
    return primes;
}
```

- MillerRabin算法

Miller-Rabin是概率性素数测试算法，用于大数的素性判断。

``` c++
using ll = long long;

ll mod_pow(ll a, ll b, ll mod) {
    ll res = 1;
    a %= mod;
    while (b > 0) {
        if (b & 1) res = (res * a) % mod;
        a = (a * a) % mod;
        b >>= 1;
    }
    return res;
}

bool check_composite(ll n, ll a, ll d, int s) {
    ll x = mod_pow(a, d, n);
    if (x == 1 || x == n - 1) return false;
    for (int r = 1; r < s; ++r) {
        x = (x * x) % n;
        if (x == n - 1) return false;
    }
    return true;
}

bool miller_rabin(ll n) {
    if (n < 2) return false;
    int s = 0;
    ll d = n - 1;
    while (d % 2 == 0) {
        d /= 2;
        ++s;
    }
    vector<ll> witnesses = {2, 3, 5, 7, 11, 13, 23, 29, 31, 37}; // for n < 2^64
    for (ll a : witnesses) {
        if (n == a) return true;
        if (n % a == 0) return false;
        if (check_composite(n, a, d, s)) return false;
    }
    return true;
}
```

#### 唯一分解定理

任何大于1的正整数都可以唯一地分解为素数的乘积（不考虑顺序）。

### 费马小定理

费马小定理：如果p是素数，且a不是p的倍数，则

$a^{p-1} \equiv 1 \pmod{p}$

#### 剩余系

剩余系是指模m的完全剩余系，即从0到m-1的所有整数。

#### 费马小定理的证明

费马小定理的证明可以使用剩余系的思想。

考虑模p下的剩余系：1, 2, ..., p-1。

将这些数乘以a（模p），得到a, 2a, ..., (p-1)a。

由于p是素数，这些数两两不同，且不等于0（因为a不是p的倍数）。

因此，${a, 2a, ..., (p-1)a} \equiv {1, 2, ..., p-1} \pmod{p}$。

将这些数相乘：

左边：$a \cdot 2a \cdot \ldots \cdot (p-1)a = a^{p-1} \cdot (1 \cdot 2 \cdot \ldots \cdot (p-1))$

右边：$1 \cdot 2 \cdot \ldots \cdot (p-1)$

所以，$a^{p-1} \cdot (p-1)! \equiv (p-1)! \pmod{p}$

由于$(p-1)! \not\equiv 0 \pmod{p}$，可以消去，得

$a^{p-1} \equiv 1 \pmod{p}$

#### 欧拉定理

欧拉函数$\varphi(n)$定义为小于或等于$n$的正整数中与$n$互素的数的个数。

欧拉函数的求法：如果$n$的素因子分解为$n = p_1^{k_1} p_2^{k_2} \ldots p_m^{k_m}$，则$\varphi(n) = n (1 - 1/p_1) (1 - 1/p_2) \ldots (1 - 1/p_m)$。

如果a和n互素，则$a^{\varphi(n)} \equiv 1 \pmod{n}$。

### 线性同余方程

线性同余方程介绍

形如$ax \equiv b \pmod{m}$的方程称为线性同余方程。

#### 裴蜀定理

对于任意整数a,b,d，如果d=gcd(a,b)，则存在整数x,y使得$ax + by = d$。

#### 扩展欧几里得算法

扩展欧几里得算法用于求解$ax + by = \gcd(a,b)$。

``` c++
int gcd(int a, int b) {
    return b == 0 ? a : gcd(b, a % b);
}

tuple<int, int, int> extended_gcd(int a, int b) {
    if (b == 0) {
        return {a, 1, 0};
    }
    auto [g, x1, y1] = extended_gcd(b, a % b);
    int x = y1;
    int y = x1 - (a / b) * y1;
    return {g, x, y};
}
```

#### 逆元的定义与性质

在模m下，a的逆元是满足$a \cdot x \equiv 1 \pmod{m}$的x。逆元存在当且仅当a和m互素，即gcd(a, m) = 1。

逆元的求法

逆元可以通过扩展欧几里得算法求得，因为它可以求解$ax + my = 1$，其中x就是a的逆元。

``` c++
// 使用扩展欧几里得求逆元
int mod_inverse(int a, int m) {
    auto [g, x, y] = extended_gcd(a, m);
    if (g != 1) {
        // 无逆元
        return -1;
    }
    return (x % m + m) % m;
}
```

使用费马小定理求逆元

当模数m是素数时，可以使用费马小定理来求逆元。费马小定理指出：如果p是素数，且a不是p的倍数，则

$a^{p-1} \equiv 1 \pmod{p}$

因此，a的逆元为

$a^{p-2} \pmod{p}$

原理：由费马小定理，a * a^{p-2} ≡ 1 (mod p)，所以a^{p-2}就是a的逆元。

``` c++
// 使用费马小定理求逆元（m为素数）
ll mod_inverse_fermat(ll a, ll m) {
    return mod_pow(a, m - 2, m);
}
```

#### 中国剩余定理

中国剩余定理用于求解模数两两互素的同余方程组。例如，求解$x \equiv a_1 \pmod{m_1}$，$x \equiv a_2 \pmod{m_2}$，...,$x \equiv a_k \pmod{m_k}$，其中m1, m2, ..., mk两两互素。

原理：设$M = m_1 \cdot m_2 \cdot \ldots \cdot m_k$，$M_i = M / m_i$，则$x = \sum (a_i \cdot M_i \cdot \text{inv}(M_i, m_i)) \pmod{M}$。
M
$\text{inv}(a, b)$表示 a 在模 b 的意义下的逆元。

中国剩余定理的题解

首先计算M = 所有模数的乘积。然后对于每个方程，计算Mi = M / mi，并求Mi在模mi下的逆元。最后按公式计算x。

``` c++
using ll = long long;

// 中国剩余定理实现
ll crt(const vector<ll>& a, const vector<ll>& m) {
    int n = a.size();
    ll M = 1;
    for (ll mod : m) M *= mod;
    ll res = 0;
    for (int i = 0; i < n; ++i) {
        ll Mi = M / m[i];
        ll inv = mod_inverse(Mi, m[i]);
        res = (res + a[i] * Mi % M * inv % M) % M;
    }
    return res;
}
```

## 例题

### CodeForces 237C: Primes on Interval

**题意**：给定正整数$a, b, k$($1 \leq a \leq b \leq 10^6$,$1 \leq k \leq b-a+1$)，找最小长度$l$($1 \leq l \leq b-a+1$)，使得在区间$[a, b]$的任何连续$l$个数中，至少有$k$个素数。如果不存在，输出$-1$。

**题解**：需要找到最小的$l$，保证每个长度为$l$的连续段至少有$k$个素数，可以用二分搜索$l$，然后检查每个可能的窗口。二分$l$的范围$[1, b-a+1]$，对于每个$mid$，检查是否存在连续$mid$个数中素数少于$k$。如果所有检查通过，则可行。检查时，预处理$[a,b]$内的素数，使用埃拉托斯特尼筛法，然后滑动窗口统计素数个数。时间复杂度$O((b-a+1) \log(b-a+1))$。

### CodeForces 385C: Bear and Prime Numbers

**题意**：给定序列$x_1$到$x_n$($2 \leq x_i \leq 10^7$,$n \leq 10^6$)，和$m$个查询，每个查询$[l,r]$($2 \leq l \leq r \leq 2\cdot10^9$)，对于每个查询，计算区间$[l,r]$内所有素数$p$的$f(p)$之和，其中$f(p)$是序列中被$p$整除的元素个数。

**题解**：$f(p)$定义为序列中$p$的倍数个数，查询需要对$[l,r]$内的$p$求和。由于$p$可以很大，需要预处理所有$x$的因子。使用哈希表记录每个$p$的$f(p)$，然后对查询使用离线处理和前缀和。时间复杂度$O(n \log n + m \sqrt{n})$。

### CodeForces 818E: Card Game Again

**题意**：给定$n$张牌，数字$a_1$到$a_n$($1 \leq a_i \leq 10^9$)，和$k$($1 \leq k \leq 10^9$)，移除$x$张顶牌和$y$张底牌后，剩余牌的乘积能被$k$整除，求$(x,y)$的对数，$x,y \geq 0$，且剩余至少一张牌。

**题解**：剩余牌是连续段，积被$k$整除意味着积包含$k$的所有素因子。计算前缀积和后缀积，找到积能被$k$整除的子段数量。使用二分查找或直接计算积的GCD。时间复杂度$O(n \log n)$。

### CodeForces 7C: Line

**题意**：给定$A, B, C$($-2\cdot10^9 \leq A,B,C \leq 2\cdot10^9$,$A^2 + B^2 > 0$)，找整数$x,y$($-5\cdot10^{18} \leq x,y \leq 5\cdot10^{18}$) ，满足$Ax + By + C = 0$。如果不存在，输出$-1$。

**题解**：方程$Ax + By = -C$，转换为同余方程，解是否存在整数解。如果$B=0$，检查$A$和$C$；否则，使用扩展欧几里得求解。时间复杂度$O(\log \max(|A|,|B|))$。

### CodeForces 1061C: Multiplicity

**题意**：给定数组$a_1$到$a_n$($1 \leq a_i \leq 10^6$,$n \leq 10^5$)，求非空子序列$b_1,b_2,\ldots,b_k$，其中$b_i \% i == 0$，子序列按位置索引不同算不同，答案$\mod 10^9+7$。

**题解**：子序列需要满足$b_i \% i == 0$，且$i$是位置索引。使用DP，$dp[i]$表示以第$i$个元素结尾的好子序列数。初始化$dp[1]=1$，对于每个$i$，从$1$到$i-1$，如果$a[j] \% j == 0$且$j \% i == 0$，则$dp[i] += dp[j]$。时间复杂度$O(n^2)$。

### CodeForces 348B: Apple Tree

**题意**：树有$n$个节点($2 \leq n \leq 10^5$)，叶子有苹果数$a[i]$($0 \leq a[i] \leq 10^8$)，非叶子$a[i]=0$。平衡树使每个节点的子树权重相等，权重为叶子苹果和。求最小移除苹果数。

**题解**：平衡意味着所有子树权重相等，等于叶子权重GCD的倍数。DFS计算每个子树权重，统计所有叶子权重$d$，求GCD$d$，然后$移除总数 - (叶子数 \cdot d)$。时间复杂度$O(n)$。

### CodeForces 1334E: Divisor Paths

**题意**：给定$D$($1 \leq D \leq 10^{15}$) ，构建约数图，边条件$x>y$,$x\%y==0$,$x/y$是素数，权重为$x$约数数减$y$约数数。$q$个查询($1 \leq q \leq 3\cdot10^5$)，求$v$到$u$的最短路径数$\mod 998244353$。

**题解**：根据观察可以大胆猜想一定会经过u和v的最小公因数，图是约数关系，路径对应素因子分解。分解$D$的素因子，DP计算每个约数的路径数。时间复杂度$O(D^{1/3} + q)$。

### 洛谷 P3868: [TJOI2009] 猜数字

**题意**：给定$n$($1 \leq n \leq 10$)，$a_1$到$a_n$,$m_1$到$m_n$($1 \leq a_i,m_i \leq 10^9$)，$m_i$两两互素，找最小正整数$x$满足$x \equiv a_i \mod m_i$。

**题解**：多个同余方程，$m_i$互素，使用中国剩余定理合并。逐步合并同余方程。时间复杂度$O(n \log M)$。
